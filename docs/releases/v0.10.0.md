# Koalafrog HQ v0.10.0

Released: 2026-07-18

## Added

- Product Studio workspace
- Beard Oil workflow
- Make Something Today
- Design a Product
- Explore Ingredients
- Follow a Proven Formula
- Product Studio rule engine
- Product Concepts
- Purchase Plan handoff
- Formula Draft handoff
- Lab handoff
- Inventory, equipment, and packaging readiness
- Predicted versus observed product profiles
- Scent direction builder

## Changed

- Shifted the main workflow from record-first to product-first
- Improved Ingredient workflow
- Simplified Supplier Product UX with progressive disclosure
- Added inherited Ingredient context
- Added category-aware grade suggestions
- Improved persistence error reporting for structured Supabase errors

## Fixed

- Product Studio concept persistence integration test could fail because it reused a globally unique test ID
- Structured Supabase errors were previously replaced by a generic persistence message
- Product Studio purchase-plan workflow now has stronger hydration, idempotency, linkage, and no-stock-write coverage

## Known limitations

- Product Studio currently supports Beard Oil only
- Ingredient Reference Library is incomplete
- Recommendations are predictions and require physical testing
- More product types are planned
- Storage file binaries require a separate authenticated export
- Production bundle-size warning remains non-blocking

## Release standard

This release is preserved only after testing, verification, backup, documentation, commit, and release tagging.
